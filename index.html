桐生あんずろぐ
プログラミングが趣味の京都の大学生です。

About
桐生あんずというハンドルネームで活動しています。
立命館大学の文系学部所属ですが、2016/10~2017/2に行われた「1ヶ月でWebサービスを作る会」と2017/3/10,11開催の「RailsGirlsKyoto7th」がきっかけでRubyを知りプログラミングの勉強を始めました。
現在は大学に通いつつ、エンジニアのアルバイト(Rails開発)やシェアハウスを拠点にプログラミング勉強会の運営を行っています。
Product
    Web Application
  • 布教箱

    • 主な使用技術: Ruby on Rails / Postgresql / Heroku / SendGrid ソースコード

    開発人数: 1人(自分)

    自分が好きなアニメや漫画のジャンル、日常生活のアドバイスなど「布教してほしいこと」をお題として投稿し、そのお題を見たユーザーが匿名で布教を投稿できる質問&回答型Webサービスです。

  • Crash Chat

    • 主な使用技術: Ruby on Rails / Postgresql / Watson ソースコード

    開発人数: 1人(自分)

    IBM開発のAPI「Watson」を利用して、オタサーの姫やオタクの人工無脳と会話して遊ぶことのできるWebアプリケーションです。

  • サークルクラッシャー診断

    • 主な使用技術: jQuery ソースコード

    開発人数: 3人(企画マネージャー&コーディング(自分) / イラストレーター / 診断文作成者)

    サークルクラッシュ同好会のNF会誌に寄稿した作品です。サークルクラッシュを京都大学で研究している方の監修の元、作られた性格診断ゲームです。8分類に分けて回答者がどのようなタイプのサークルクラッシャーであるかを診断します。

  • dot.para

    • 使用技術: Processing ソースコード

    開発人数: 2~3人(企画(自分) / 既存会員とのペアプロ開発)

    京大マイコンクラブで開催された「ラピッドコーディング祭り(2018/5/2)」で既存会員の方と一緒に製作したゲームです。アイドルの女の子が飛んでくるいいねを集めてスコアを稼ぐゲームです。
